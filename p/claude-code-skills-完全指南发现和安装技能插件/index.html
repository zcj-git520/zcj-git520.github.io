Claude Code Skills 完全指南：发现和安装技能插件

Apr 30, 2026
阅读时长: 2 分钟

Claude Code 的 Skills 生态系统让 AI 具备专业化能力。本文将带你了解如何发现和安装 Skills，提升 AI 编程效率。

一、什么是 Skills CLI？

Skills CLI（npx skills）是开放技能生态系统的包管理器。Skills 是模块化包，为 AI Agent 扩展专业能力。

核心命令

  • npx skills find [query] - 交互式搜索或关键词搜索
  • npx skills add <package> - 从 GitHub 等来源安装技能
  • npx skills check - 检查技能更新
  • npx skills update - 更新所有已安装技能

浏览技能：https://skills.sh/

二、何时使用 Find Skills 技能？

当用户表达以下需求时使用：

  • 问"如何做 X"（X 可能是常见任务）
  • 说"找一个做 X 的技能"
  • 问"你能做 X 吗"（X 是专业能力）
  • 想扩展 AI 能力
  • 想搜索工具、模板或工作流

三、使用步骤

步骤 1：了解用户需求

识别：

  1. 领域 - 例如 React、测试、设计、部署
  2. 具体任务 - 例如写测试、创建动画、审查 PR
  3. 是否常见任务 - 是否有现成的技能

步骤 2：先查 Leaderboard

先查看 skills.sh leaderboard 找出热门技能。

例如，Web 开发的热门技能：

  • vercel-labs/agent-skills — React、Next.js、Web 设计（100K+ 安装）
  • anthropics/skills — 前端设计、文档处理（100K+ 安装）

步骤 3：搜索技能

如果 leaderboard 没有覆盖需求，运行搜索：

npx skills find [query]

示例：

  • “如何让 React 应用更快？” → npx skills find react performance
  • “能帮我审查 PR 吗？” → npx skills find pr review
  • “需要创建 changelog” → npx skills find changelog

步骤 4：验证质量

不要仅根据搜索结果推荐技能。验证：

  1. 安装数 - 优先选择 1K+ 安装的技能，100 以下需谨慎
  2. 来源声誉 - 官方来源（vercel-labs、anthropics、microsoft）更可靠
  3. GitHub stars - 检查源仓库，<100 stars 需要怀疑

步骤 5：向用户展示

展示找到的技能：

我找到了一个可能有用的技能！"react-best-practices" 提供来自
Vercel Engineering 的 React 和 Next.js 性能优化指南。

安装命令：
npx skills add vercel-labs/agent-skills@react-best-practices

了解更多：https://skills.sh/vercel-labs/agent-skills/react-best-practices

步骤 6：协助安装

如果用户同意，可以帮忙安装：

npx skills add <owner/repo@skill> -g -y

-g 全局安装，-y 跳过确认。

四、常见技能分类

分类 示例查询
Web 开发 react, nextjs, typescript, css, tailwind
测试 testing, jest, playwright, e2e
DevOps deploy, docker, kubernetes, ci-cd
文档 docs, readme, changelog, api-docs
代码质量 review, lint, refactor, best-practices
设计 ui, ux, design-system, accessibility
生产力 workflow, automation, git

五、搜索技巧

  1. 使用具体关键词 - “react testing” 比 “testing” 更好
  2. 尝试替代词 - 如果 “deploy” 不行，试试 “deployment” 或 “ci-cd”
  3. 检查热门来源 - 很多技能来自 vercel-labs/agent-skills

六、当没有找到技能时

如果没找到相关技能：

  1. 承认没有找到匹配技能
  2. 提供直接帮助
  3. 建议用户可以自己创建：npx skills init

示例：

我搜索了"xyz"相关的技能，但没有找到匹配项。
我可以直接帮你完成这个任务！想让我继续吗？

如果这是你经常做的，可以创建自己的技能：
npx skills init my-xyz-skill

nextjs Next.js 开发 npx skills add vercel-labs/agent-skills@nextjs

技能 功能 安装命令
testing 测试指南 npx skills add anthropics/skills@testing
playwright E2E 测试 npx skills add vercel-labs/agent-skills@playwright

文档

技能 功能 安装命令
documentation 文档生成 npx skills add anthropics/skills@documentation
readme README 生成 npx skills add vercel-labs/agent-skills@readme

八、总结

使用 Find Skills 技能：

  1. 先理解需求 - 明确领域和具体任务
  2. 查热门榜 - leaderboard 上的技能更可靠
  3. 搜索验证 - 用具体关键词搜索
  4. 验证质量 - 看安装数和来源
  5. 展示安装 - 提供完整信息

Skills 生态系统让 Claude Code 变得更强大，帮你完成各种专业任务。
